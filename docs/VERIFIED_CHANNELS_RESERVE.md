## Инструкция по использованию

### 1. Перед добавлением канала ОБЯЗАТЕЛЬНО проверь:

1. Открой Telegram
2. Найди канал по username
3. Проверь что канал СУЩЕСТВУЕТ
4. Проверь что комментарии ОТКРЫТЫ (есть поле ввода под постами)
5. Только потом добавляй в систему

### 2. Добавление в систему (по 5-10 каналов):

```bash
cd "c:\Users\mafio\OneDrive\Документы\projects\info-business\traffic-engine-mvp"
python scripts\add_target_channels.py
```

### 3. Безопасность:

- НЕ подписываться на каналы с рабочих аккаунтов
- Добавлять постепенно (5-10 каналов за раз)
- Не больше 5 комментов/день в первую неделю
- Прогрев: 5 → 10 → 25 → 50 комментов/день
