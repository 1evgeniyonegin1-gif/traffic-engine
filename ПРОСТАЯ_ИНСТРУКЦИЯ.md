## ПОЧЕМУ ТАК СЛОЖНО?

Auth Keys с сервиса покупки привязаны к другому IP адресу. Telegram не разрешает использовать один auth_key с разных IP одновременно. Поэтому нужно создать НОВЫЕ сессии прямо на вашем сервере.

Это нужно сделать только ОДИН РАЗ. Потом всё будет работать автоматически.
